Welcome!

Thanks for using Excalibur!

If you have questions, don't hesitate to reach out to us on our Google Groups or open an issue on GitHub.

Get Excalibur

Bower: bower install excalibur

NPM: npm install excalibur

NuGet: Install-Package Excalibur

Latest version on GitHub: https://github.com/excaliburjs/Excalibur

Including Excalibur

<!DOCTYPE html>
<html lang="en">
    <head>
    </head>
    <body>
        <script src="excalibur-version.js"></script>
    </body>
</html>

You're all done!
