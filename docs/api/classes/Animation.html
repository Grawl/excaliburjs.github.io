ex.Animation

src\engine\Drawing.ts

Inheritance:
  • IDrawable
  •     Animation
Animations allow you to display a series of images one after another, creating the illusion of change. Generally these images will come from a sprite sheet source.

Constructor

new ex.Animation(engine: Engine, images: Sprite[], speed: Number, loop?: Boolean)
engine Engine Reference to the current game engine
images Sprite[] An array of sprites to create the frames for the animation
speed Number The number in milliseconds to display each frame in the animation
loop? = false Boolean Indicates whether the animation should loop after it is completed
This method has no description
