ex.IDrawable

src\engine\Drawing.ts

Interface for implementing anything in Excalibur that can be drawn to the screen.

Properties

Name Type Description
flipHorizontal Boolean Indicates whether the drawing is to be flipped horizontally
flipVertical Boolean Indicates whether the drawing is to be flipped vertically
height Number Indicates the height of the drawing in pixels
width Number Indicates the width of the drawing in pixels

Methods

addEffect(effect: ISpriteEffect)
effect ISpriteEffect Effect to add to the this drawing
Adds a new to this drawing.
clearEffects()
Clears all effects from the drawing and return it to its original state.
draw(ctx: CanvasRenderingContext2D, x: Number, y: Number)
ctx CanvasRenderingContext2D The 2D rendering context
x Number The x coordinate of where to draw
y Number The y coordinate of where to draw
Draws the sprite appropriately to the 2D rendering context.
getRotation(): number
Returns the current rotation for the drawing.
getScaleX(): number
Returns the current magnitude of the drawing's scale in the x direction.
getScaleY(): number
Returns the current magnitude of the drawing's scale in the y direction.
reset()
Resets the internal state of the drawing (if any)
setRotation(radians: Number)
radians Number The rotation to apply to the drawing.
Sets the current rotation transformation for the drawing.
setScale(scale: Number)
scale Number The magnitude to scale the drawing in the x direction
Sets the scale trasformation
setScale(scale: Number)
scale Number The magnitude to scale the drawing in the y direction
Sets the scale trasformation
transformAbotPoint(point: Point)
point Point The point about which to apply transformations
Sets the point about which to apply transformations to the drawing relative to the top left corner of the drawing.
