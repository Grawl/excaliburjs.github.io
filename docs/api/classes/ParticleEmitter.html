ex.ParticleEmitter

src\engine\Particles.ts

Using a particle emitter is a great way to create interesting effects in your game, like smoke, fire, water, explosions, etc. Particle Emitters extend Actor allowing you to use all of the features that come with Actor

Constructor

new ex.ParticleEmitter(x?: Number, y?: Number, width?: Number, height?: Number)
x? = 0 Number The x position of the emitter
y? = 0 Number The y position of the emitter
width? = 0 Number The width of the emitter
height? = 0 Number The height of the emitter
This method has no description
