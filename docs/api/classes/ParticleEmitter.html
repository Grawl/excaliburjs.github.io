ex.ParticleEmitter

Using a particle emitter is a great way to create interesting effects in your game, like smoke, fire, water, explosions, etc. Particle Emitters extend Actor allowing you to use all of the features that come with Actor

Constructor

new ex.ParticleEmitter(x? : Number, y? : Number, width? : Number, height? : Number)
Name Type Description
x? = 0 Number The x position of the emitter
y? = 0 Number The y position of the emitter
width? = 0 Number The width of the emitter
height? = 0 Number The height of the emitter

Properties

Name Type Description
isEmitting Boolean Gets or sets the isEmitting flag
particles Util.Collection<Particle> Gets or sets the backing particle collection
particles Util.Collection<Particle> Gets or sets the backing deadParticle collection
[minVel=0] Number Gets or sets the minimum partical velocity
[maxVel=0] Number Gets or sets the maximum partical velocity
[acceleration=new Vector(0,0)] Vector Gets or sets the acceleration vector for all particles
[minAngle=0] Number Gets or sets the minimum angle in radians
[maxAngle=0] Number Gets or sets the maximum angle in radians
[emitRate=1] Number Gets or sets the emission rate for particles (particles/sec)
[particleLife=2000] Number Gets or sets the life of each particle in milliseconds
[opacity=1.0] Number Gets or sets the opacity of each particle from 0 to 1.0
[fade=false] Boolean Gets or sets the fade flag which causes particles to gradually fade out over the course of their life.
[focus=null] Vector Gets or sets the optional focus where all particles should accelerate towards
[focusAccel=1] Number Gets or sets the acceleration for focusing particles if a focus has been specified
[minSize=5] Number Gets or sets the minimum size of all particles
[maxSize=5] Number Gets or sets the maximum size of all particles
[beginColor=Color.White] Color Gets or sets the beginning color of all particles
[endColor=Color.White] Color Gets or sets the ending color of all particles
[particleSprite=null] Sprite Gets or sets the sprite that a particle should use
[emitterType=EmitterType.Rectangle] EmitterType Gets or sets the emitter type for the particle emitter
[radius=0] Number Gets or sets the emitter radius, only takes effect when the emitterType is Circle

Methods

emit(particleCount : Number)
Causes the emitter to emit particles
Name Type Description
particleCount Number Number of particles to emit right now
