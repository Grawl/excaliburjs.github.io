ex.Polygon

Extends: IDrawable

Creates a closed polygon drawing given a list a of points. Polygons should be used sparingly as there is a performance impact for using them.

Constructor

new ex.Polygon(points : Point[])
Name Type Description
points Point[] The points to use to build the polygon in order

Properties

Name Type Description
lineColor Color The color to use for the lines of the polygon
fillColor Color The color to use for the interior of the polygon
[lineWidth=5] The width of the lines in pixels Number The width of the lines of the polygon
[filled=false] Boolean Indicates whether the polygon is filled or not.
