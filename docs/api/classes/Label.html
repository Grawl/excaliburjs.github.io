ex.Label

Extends: Actor

Labels are the way to draw small amounts of text to the screen in Excalibur. They are actors and inherit all of the benifits and capabilities.

Constructor

new ex.Label(text? : String, x? : Number, y? : Number, font? : String, spriteFont? : SpriteFont)
Name Type Description
text? = empty String The text of the label
x? = 0 Number The x position of the label
y? = 0 Number The y position of the label
font? = sans-serif String Use any valid css font string for the label's font. Default is "10px sans-serif".
spriteFont? = undefined SpriteFont Use an Excalibur sprite font for the label's font, if a SpriteFont is provided it will take precendence over a css font.

Properties

Name Type Description
textAlign TextAlign Gets or sets the horizontal text alignment property for the label.
textBaseline BaseAlign Gets or sets the baseline alignment property for the label.
maxWidth Number Gets or sets the maximum width (in pixels) that the label should occupy
[letterSpacing=0] Number Gets or sets the letter spacing on a Label. Only supported with Sprite Fonts.
