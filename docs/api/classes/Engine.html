ex.Engine

src\engine\Core.ts

The 'Engine' is the main driver for a game. It is responsible for starting/stopping the game, maintaining state, transmitting events, loading resources, and managing the scene.

Constructor

new ex.Engine(width?: Number, height?: Number, canvasElementId?: String, displayMode?: DisplayMode)
width? Number The width in pixels of the Excalibur game viewport
height? Number The height in pixels of the Excalibur game viewport
canvasElementId? String If this is not specified, then a new canvas will be created and inserted into the body.
displayMode? DisplayMode If this is not specified, then it will fall back to fixed if a height and width are specified, else the display mode will be FullScreen.
This method has no description
