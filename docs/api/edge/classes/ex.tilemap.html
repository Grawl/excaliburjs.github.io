Class TileMap

The CollisionMap object provides a lightweight way to do large complex scenes with collision without the overhead of actors.

class

CollisionMap

constructor
param

The x coordinate to anchor the collision map's upper left corner (should not be changed once set)

param

The individual width of each cell (in pixels) (should not be changed once set)

param

The individual height of each cell (in pixels) (should not be changed once set)

param

The number of rows in the collision map (should not be changed once set)

param

The number of cols in the collision map (should not be changed once set)

param

The spriteSheet to use for drawing
